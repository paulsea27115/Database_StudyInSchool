## DataBase 용어
- relation = table
- 튜플
- 속성(Attribute) - degree
- 카디널리티
- 도메인 - 값의 범위
- 인스턴스 - 계속 바뀜
- key - 바뀔수 있는 속성
- 슈퍼키 - 모든 키(각각의 튜플 을 부분짓는 속성들의 집합 = 유일성)
- 후보키 - 가장 작은 수의 키 (슈퍼키중 길이가 잛은 집합 = 최소성, 유일성)
- 기본키 - 후보키중 하나 (후보키중 DBA가 선택한 집합)
- 대체키 - 기본키를 제외한 다른 후보키들 (후보키 - 기본키) 
##### 후보키 = 키본키 + 대체키
- 대리키 - 의미없는 속성을 기본키로 만든 키
- 외래키 - 다른 테이블의 기본키가 내 테이블에 들어있는 키
- 단일키
- 복합키
